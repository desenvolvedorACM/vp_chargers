## Running the project

### Run Expo Environment

```
npx expo start / yarn start
```

### Run Native Development

**Install dependencies**

```
npm / yarn
```

**Install IOS Pods**

```
cd ios && pod install
```

**Run IOS**

```
yarn ios
```

**Run Android**

```
yarn android
```

## Running the tests

Use **yarn test** to execute the jest tests.

```cl
yarn test
```
